`@polo/core` exports a small runtime API and supporting types.

## Runtime functions

```ts
import { createPolo, registerSources } from "@polo/core";
```

- `createPolo(options?)` creates a runtime instance.
- `registerSources(...sourceSets)` merges reusable source sets.

## Main runtime methods

After `createPolo()`, you call:

- `polo.source.fromInput(...)`
- `polo.source(...)`
- `polo.source.chunks(...)`
- `polo.sourceSet(...)`
- `polo.define(...)`
- `polo.resolve(...)`

## Common types

```ts
import type {
  InferContext,
  Trace,
  BudgetConfig,
  BudgetStrategyFn,
  Chunk,
} from "@polo/core";
```
